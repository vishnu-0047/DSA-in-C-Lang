# Doubly Linked List in C

## Overview
This project implements a **Doubly Linked List** in C with a **menu-driven interface**, allowing users to perform insertion, deletion, traversal, and search operations.  

It demonstrates:
- Dynamic memory allocation using `malloc()` and `free()`
- Pointers and structures
- CRUD operations on a doubly linked list
- Console-based user interaction

---

## Features

### Insertion
- At the beginning
- At the end
- At a specific position

### Deletion
- From the beginning
- From the end
- From a specific position

### Other Operations
- Search for value at a specific position
- Display the linked list with indices

---

## Tech Stack
- Language: C
- Platform: Any C compiler (GCC, Code::Blocks, Dev-C++, etc.)

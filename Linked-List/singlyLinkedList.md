# Singly Linked List in C

## Description
This project implements a **Singly Linked List** in C, providing standard operations such as insertion, deletion, traversal, and search. The program includes a **menu-driven interface** for easy interaction, allowing users to manipulate the linked list dynamically.  

It demonstrates the following concepts:  
- Dynamic memory allocation using `malloc()` and `free()`  
- Pointers and node structures  
- CRUD operations on a linked list  
- Menu-driven console interaction  

---

## Features

- **Insertion**:  
  - At the beginning  
  - At the end  
  - At a specific position  

- **Deletion**:  
  - From the beginning  
  - From the end  
  - From a specific position  

- **Search**: Find value at a specific position  
- **Traversal**: Display the linked list and corresponding indices  

---

## Tech Stack
- **Language:** C  
- **Platform:** Any C compiler (GCC, Code::Blocks, Dev-C++, etc.)

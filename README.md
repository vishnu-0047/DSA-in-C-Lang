# Data Structures & Algorithms Practice

## Description
Contains collection of DSA problems. Includes implementation of binary search tree, linked lists, queues, sorting algorithms, and more. All code is written in C.

## Tech Stack
- C

## Folder Structure
- Binary Search Tree: Creation of BST, In-order, Pre-order, Post-order traversal, Insertion, Deletion, Searching Operations
- Linked List: Singly/doubly linked list implementations, CIrcular linked list implementation
- Queues: Linear & Circular Queue implementations, Round Robin Scheduling using Circular Queues
- Heap: Min heap and max heap structures
- Sorting: Quick Sort, Merge Sort

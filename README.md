# Data Structures & Algorithms Practice

## Description
Contains collection of DSA problems. Includes implementation of arrays, linked lists, stacks, queues, sorting algorithms, and more. All code is written in C.

## Tech Stack
- Java
- C

## Folder Structure
- Arrays: Problems like sum, reverse, search
- LinkedList: Singly/doubly linked list implementations
- Stacks_Queues: Stack & Queue implementations
- Sorting: Bubble, Selection, Insertion, Quick Sort
